A modern, high-speed, and secure self-bot tool designed to help you manage your digital footprint by purging your messages from specific Discord channels or DMs.

## ‚ú® Features

* **Modern Dark UI:** Built with a sleek, professional interface using CustomTkinter.
* **Real-Time Terminal:** Live logging of every message deleted so you can track progress.
* **Safe Deletion:** Built-in rate-limit handling and a minimum 1.5s delay to protect your account from bans.
* **Multi-Threaded:** The app never freezes while the purge is running.



### **How to get your Token**
To use this tool, you need your Discord Authorization Token. Follow the visual guide below:
1. Open Discord in your browser.
2. Press `F12` to open Developer Tools.
3. Go to the **Network** tab and reload the page.
4. Filter for `authorization`.
